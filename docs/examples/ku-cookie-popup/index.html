KU cookie popup

How to implement KU cookie popup on external sites.

  1. Add the stylesheet in <head>:
    https://cms.ku.dk/bootstrap/dist/css/ku-cookiepopup.min.css
  2. Add the following scripts in <head>:

    <script>KUCookieFilters.output_popup_placeholder();</script>

    <script src="https://cms.ku.dk/scripts_js/cookies/cookie_filters.js?f21d267"></script>

  3. Add the following scripts right before </body>:

    <script src="https://cms.ku.dk/scripts_js/cookies/cookiedata.js?f21d267"></script>

    <script>KUCookieFilters.init(".someotherdomain.dk")</script>

    <script>window.GLOBAL_cookieconf.liveStatsEndpoint = "https://www.ku.dk/:obvius/consent";</script>

    <script src="https://cms.ku.dk/scripts_js/cookies/ccc.js?f21d267"></script>

  4. Wrap your custom scripts (that set statistical or marketing cookies) in the following function: 
    
    <script>
    KUCookieFilters.wrap_inline_script(function() {
      // Insert code here
    }, ["statistical", "marketing"]);
    </script>
    Add parameters "statistical" and/or "marketing" depending on the type of cookies set by the script.

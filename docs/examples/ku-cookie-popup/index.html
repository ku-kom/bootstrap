KU cookie popup

How to implement KU cookie popup on external sites.

  1. Add the following scripts in <head>:
    <script>KUCookieFilters.output_popup_placeholder();</script>
    <script src="https://cms.ku.dk/scripts_js/cookies/cookie_filters.js?f21d267"></script>
  2. Add the following scripts right before </body>:
    <script src="https://cms.ku.dk/scripts_js/cookies/cookiedata.js?f21d267"></script>
    <script>KUCookieFilters.init(".ku.dk")</script>
    <script>window.GLOBAL_cookieconf.liveStatsEndpoint = "https://www.ku.dk/:obvius/consent";</script>
    <script src="https://cms.ku.dk/scripts_js/cookies/ccc.js?f21d267"></script>
  3. If the domain is not *.ku.dk, please change the domain name here:
    <script>KUCookieFilters.init(".ku.someotherdomain.dk")</script>

Global menus

Global menus are avaliable as html snippets that will be updated automatically by KU. It can be included into external systems using serverside code such as php, java, etc. in order to keep menus consistent with the rest of the KU domain.

The urls are:

Top menu

Danish: https://cms.ku.dk/menu-spippets/global-topmenu.html

English: https://cms.ku.dk/menu-spippets/eng-global-topmenu.html

Second level top menu

Danish: https://cms.ku.dk/menu-spippets/global-second-level-menu.html

English: https://cms.ku.dk/menu-spippets/eng-global-second-level-menu.html

Footer

Danish: https://cms.ku.dk/menu-spippets/global-footer.html

English: https://cms.ku.dk/menu-spippets/eng-global-footer.html

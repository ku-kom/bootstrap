KU faculty specific styling

In order to display a page with the colors of a certain faculty, a class must be added to the <body> tag.

For example, <body theme-science> will by default produce link colors, branding, etc. relative to the faculty of Science.

Possible classes are:

{% highlight html %}
  • None: Default KU styling
  • theme-samf: Faculty of Social Sciences
  • theme-sund: Faculty of Health Sciences
  • theme-science: Faculty of Science
  • theme-jura: Faculty of Law
  • theme-teol: Faculty of Theology
  • theme-hum: Faculty of Humanities
{% endhighlight %}
